# 📦 Antigravity Coffee Order System - 배포 패키지

이 디렉토리는 **블랙업 커피 로스터스 원두 주문 관리 시스템**의 실행 파일과 설정, 가이드 문서가 모두 포함된 **통합 패키지**입니다.

---

## 1. 패키지 구성 (Directory Structure)

이 폴더(`blackup_bean_order`)를 통째로 로스터(운영자)의 컴퓨터에 복사하세요.

- **`.agent/`**: 🤖 **[핵심]** 에이전트의 두뇌(Instruction)와 스킬 설정이 들어있습니다. (절대 수정/삭제 금지)
- **`src/`**: 🧠 **[핵심]** 주문 검증 및 처리 로직이 담긴 Python 코드입니다. (`validator.py` 등)
- **`config/`**: ⚙️ `settings.yaml` 및 `service_account.json` (인증키)이 위치합니다.
- **`reports/`**: 📄 매일 생성되는 주문 리포트가 저장되는 곳입니다.
- **`templates/`**: 📝 구글 시트 양식과 리포트 템플릿이 들어있습니다.
- **`docs/`**: 📚 각종 매뉴얼과 가이드 문서입니다.
  - `docs/handover/`: **[중요]** 사용자별(로스터, 점장) 전달용 문서가 모여 있습니다.

---

## 2. 설치 및 실행 (Installation)

로스터의 PC에서 다음 단계를 수행하세요.

1.  **Antigravity Agent 설치**: (사전 설치 필요)
2.  **폴더 배치**: 이 폴더를 `바탕화면`이나 `문서` 폴더 등 접근하기 쉬운 곳에 둡니다.
3.  **인증 설정**: `config/` 폴더에 구글 서비스 계정 키(`service_account.json`)를 넣습니다.
4.  **실행 테스트**:
    - 에이전트를 켜고, 채팅창에 **"오늘 주문 정리해줘"**라고 입력해봅니다.
    - 에이전트가 백그라운드에서 데이터를 수집하고 리포트를 생성하는지 확인합니다.

---

## 3. 사용자별 전달 가이드 (Handover Guide)

각 담당자에게 **무엇을** 전달해야 하는지 정리했습니다.

### 👤 로스터 (시스템 운영자)
- **전달물**: 이 폴더 전체 (`blackup_bean_order`)가 설치된 PC.
- **보여줄 문서**: `docs/handover/for_roaster.md` (운영 가이드)
- **핵심 메시지**: "아침마다 에이전트에게 '오늘 주문 정리해줘'라고 말하세요."

### 🏪 매장 매니저 (점장)
- **전달물**: 각 매장의 **Google Sheets URL** (카톡/메신저로 공유).
- **보여줄 문서**: `docs/handover/for_store_manager.md` (PDF나 이미지로 보내주세요)
- **핵심 메시지**: "주문은 시트에 넣고, 물건 받으면 맨 오른쪽 체크박스(☑️) 꼭 눌러주세요."

### 💻 개발자 (유지보수)
- **전달물**: 이 폴더 전체 + Git Repository 접근 권한.
- **보여줄 문서**: `docs/handover/for_developer.md` (아키텍처/코드 설명)

---

## 4. 주의사항

1.  **폴더 구조 유지**: 폴더 이름이나 파일 위치를 임의로 바꾸면 에이전트가 고장납니다.
2.  **설정 변경**: 매장이 늘거나 원두 가격이 바뀌면, 로스터가 **말로** 에이전트에게 시키면 됩니다. (`settings.yaml` 직접 수정 불필요)
